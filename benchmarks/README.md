# Kaggle Benchmark Log

This folder tracks the public run history for the standalone HNDSR research repo.

- `kaggle_runs.tsv` is the durable run ledger.
- Keep rows terse, factual, and comparable.
- Do not delete failed runs; mark them clearly.

Suggested update points:

1. after preflight
2. after Kaggle completion or crash
3. after review and promotion decision
